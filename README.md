## INTRO
모든 회원이 지식공유자, 구독자가 될 수 있으며 <br>
지식공유자와 구독자가 실시간 화상회의를 통해 지식 공유(강의)를 할 수 있고<br>
강의 종료 후 구독자는 평가에 따라 일정비율의 포인트를 환불받을 수 있는 플랫폼입니다.
